
# Organizador Automático de Archivos

Organiza los archivos automáticamente primero clasifica cada archivo segun su categoria, dentro de cada categoria los archivos se agrupan de acuerdo al mes en que fueron creados si encuentra un duplicado lo elimina automáticamente cuenta con un archivo de registro en el cuál se guardan todas las acciones del script asi puedes revisar qué pasó durante la organización.
